## 🚀 Installation

### Step 1 — Enable Zygisk

- **Magisk:** Open Magisk Manager → Settings → Toggle **Zygisk ON** → Reboot
- **KernelSU:** Install ZygiskNext module if not present → Enable → Reboot

### Step 2 — Flash the Module

1. Download `VIPWapp-v1.0.0-Magisk-KernelSU-Module.zip` from [**Releases**](https://github.com/YOUR_USERNAME/VIPWapp/releases/latest)
2. Open Magisk or KSU Manager → **Modules** → **Install from storage**
3. Select the `.zip` → wait for completion → **Reboot**

### Step 3 — Install the Companion App

> ⚠️ The module injects into WhatsApp. The APK is where you control all features.

1. Download **VIP Wapp APK** from [**Releases**](https://github.com/YOUR_USERNAME/VIPWapp/releases/latest)
2. Install the APK (enable "Install from unknown sources" if prompted)
3. Open VIP Wapp → enable the features you want → restart WhatsApp

### Step 4 — Verify (KSU Users)

Tap the **Action ▶** button on the module in KSU Manager:

## ❓ FAQ

<details>
<summary><b>Does this work without root?</b></summary>
No. VIP Wapp is a Magisk/KernelSU module and requires root with Zygisk enabled.
</details>

<details>
<summary><b>Do I need LSPosed?</b></summary>
No. VIP Wapp works completely independently via Zygisk. LSPosed is not required.
</details>

<details>
<summary><b>Will this get my WhatsApp account banned?</b></summary>
VIP Wapp includes a Bootloader Spoofer to minimize detection risk. However, any WhatsApp modification carries theoretical risk. Use at your own discretion.
</details>

<details>
<summary><b>WhatsApp updated and features broke. What do I do?</b></summary>
Wait for a VIP Wapp update compatible with the new WhatsApp version. Check the Releases page or Telegram channel for announcements.
</details>

<details>
<summary><b>Does it work with WhatsApp Business?</b></summary>
Yes. VIP Wapp supports both <code>com.whatsapp</code> (standard) and <code>com.whatsapp.w4b</code> (Business).
</details>

<details>
<summary><b>KSU Action says "INSTALL APP FROM REPO". Why?</b></summary>
The companion APK is not installed. Download and install VIP Wapp APK from the Releases page.
</details>

<details>
<summary><b>Why is the module zip small? Where are the features?</b></summary>
The module is a Zygisk injector — lightweight by design. All 100+ features live inside the companion APK, which hooks into WhatsApp at boot via the module.
